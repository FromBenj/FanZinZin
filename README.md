# FanZinZin
___
#### _Helping a friend while learning new skills_
##### Version 1.0: <a href="https://frombenj.github.io/FanZinZin/" target='_blank'>https://frombenj.github.io/FanZinZin/</a>

### What it is
___

**FanZinZin** is a **SPA web tool** designed for "Fanzine" writers, to easily generate A5 PDF templates tailored to their needs. 
These templates can be imported into Canva, where users can write their articles before sending them to their favorite printer.

### Technical challenges
___
- **pdf-lib** for PDF creation and modification
- **Quill** WYSIWYG editor
- **GitHub Pages** to deploy the app
- **Vite** build tool

### Familiar Technologies
___
- Vanilla JS
- GSAP
- CSS
- Bootstrap
- SVG
